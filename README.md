# RC4toSodiumProxy :
A simple Clash Royale proxy

Proxy work like this:<br />
        
        Client ------->   Proxy     -------> Server
               (RC4)    (Convert)   (Sodium)
        Client <-------   Proxy     <------ Server

Run it with:<br />
`python Main.py`

For more information:<br />
`python Main.py -h`
